Advanced Scientific Data Format is a proposed[1] replacement to the FITS standard for astronomical images and other astronomical data. The metadata is contained in a YAML header followed by binary or ASCII data.

Improvements to FITS[edit]

The Flexible Image Transport System FITS standard[2] is a widely used data format in astronomy that incorporates metadata and ascii or binary data in the same file. However, the FITS standard has several limitations that make it difficult to use for complicated and hierarchical data. For example, the FITS 'cards' have keywords limited to only 8 characters, which can make it difficult to properly describe the value associated with it and the value for each keyword and its comment cannot be longer than 68 characters.[1] By using YAML, more sophisticated and nested data structures may be used in ASDF than FITS.
