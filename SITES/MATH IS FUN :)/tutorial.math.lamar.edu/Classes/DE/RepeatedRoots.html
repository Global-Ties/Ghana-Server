Section 3-4 : Repeated Roots

In this section we will be looking at the last case for the constant coefficient, linear, homogeneous second order differential equations. In this case we want solutions to

\[ay'' + by' + cy = 0\]

where solutions to the characteristic equation

\[a{r^2} + br + c = 0\]

are double roots \({r_1} = {r_2} = r\).

This leads to a problem however. Recall that the solutions are

\[{y_1}\left( t \right) = {{\bf{e}}^{{r_1}\,t}} = {{\bf{e}}^{r\,t}}\hspace{0.25in}{y_2}\left( t \right) = {{\bf{e}}^{{r_2}\,t}} = {{\bf{e}}^{r\,t}}\]

These are the same solution and will NOT be “nice enough” to form a general solution. We do promise that we’ll define “nice enough” eventually! So, we can use the first solution, but we’re going to need a second solution.

Before finding this second solution let’s take a little side trip. The reason for the side trip will be clear eventually. From the quadratic formula we know that the roots to the characteristic equation are,

\[{r_{1,2}} = \frac{{ - b \pm \sqrt {{b^2} - 4ac} }}{{2a}}\]

In this case, since we have double roots we must have

\[{b^2} - 4ac = 0\]

This is the only way that we can get double roots and in this case the roots will be

\[{r_{1,2}} = \frac{{ - b}}{{2a}}\]

So, the one solution that we’ve got is

\[{y_1}\left( t \right) = {{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\]

To find a second solution we will use the fact that a constant times a solution to a linear homogeneous differential equation is also a solution. If this is true then maybe we’ll get lucky and the following will also be a solution

\[\begin{equation}{y_2}\left( t \right) = v\left( t \right){y_1}\left( t \right) = v\left( t \right){{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\label{eq:eq1}\end{equation}\]

with a proper choice of \(v(t)\). To determine if this in fact can be done, let’s plug this back into the differential equation and see what we get. We’ll first need a couple of derivatives.

\[\begin{align*}{{y'}_2}\left( t \right) & = v'\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} - \frac{b}{{2a}}v\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\\ {{y''}_2}\left( t \right) & = v''\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} - \frac{b}{{2a}}v'\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} - \frac{b}{{2a}}v'\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} + \frac{{{b^2}}}{{4{a^2}}}v\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\\ & = v''\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} - \frac{b}{a}v'\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} + \frac{{{b^2}}}{{4{a^2}}}v\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\end{align*}\]

We dropped the \(\left( t \right)\) part on the \(v\) to simplify things a little for the writing out of the derivatives. Now, plug these into the differential equation.

\[a\left( {v''\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} - \frac{b}{a}v'\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} + \frac{{{b^2}}}{{4{a^2}}}v\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}} \right) + b\left( {v'\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}} - \frac{b}{{2a}}v\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}} \right) + c\left( {v\,{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}} \right) = 0\]

We can factor an exponential out of all the terms so let’s do that. We’ll also collect all the coefficients of \(v\) and its derivatives.

\[\begin{align*}{{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\left( {av'' + \left( { - b + b} \right)v' + \left( {\frac{{{b^2}}}{{4a}} - \frac{{{b^2}}}{{2a}} + c} \right)v} \right) & = 0\\ {{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\left( {av'' + \left( { - \frac{{{b^2}}}{{4a}} + c} \right)v} \right) & = 0\\ {{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\left( {av'' - \frac{1}{{4a}}\left( {{b^2} - 4ac} \right)v} \right) & = 0\end{align*}\]

Now, because we are working with a double root we know that that the second term will be zero. Also exponentials are never zero. Therefore, \(\eqref{eq:eq1}\) will be a solution to the differential equation provided \(v(t)\) is a function that satisfies the following differential equation.

\[av'' = 0\hspace{0.25in}{\mbox{OR}}\hspace{0.25in}v'' = 0\]

We can drop the \(a\) because we know that it can’t be zero. If it were we wouldn’t have a second order differential equation! So, we can now determine the most general possible form that is allowable for \(v(t)\).

\[v' = \int{{v''\,dt}} = c\hspace{0.25in}v\left( t \right) = \int{{v'\,dt}} = ct + k\]

This is actually more complicated than we need and in fact we can drop both of the constants from this. To see why this is let’s go ahead and use this to get the second solution. The two solutions are then

\[{y_1}\left( t \right) = {{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\hspace{0.25in}{y_2}\left( t \right) = \left( {ct + k} \right){{\bf{e}}^{ - \frac{{b\,t}}{{2a}}}}\]

Eventually you will be able to show that these two solutions are “nice enough” to form a general solution. The general solution would then be the following.

\[\begin{align*}y\left( t \right) & = {c_1}{{\bf{e}}^{ - \,\frac{{b\,t}}{{2a}}}} + {c_2}\left( {ct + k} \right){{\bf{e}}^{ - \,\frac{{b\,t}}{{2a}}}}\\ & = {c_1}{{\bf{e}}^{ - \,\frac{{b\,t}}{{2a}}}} + \left( {{c_2}ct + {c_2}k} \right){{\bf{e}}^{ - \,\frac{{b\,t}}{{2a}}}}\\ & = \left( {{c_1} + {c_2}k} \right){{\bf{e}}^{ - \,\frac{{b\,t}}{{2a}}}} + {c_2}c\,t\,{{\bf{e}}^{ - \,\frac{{b\,t}}{{2a}}}}\end{align*}\]

Notice that we rearranged things a little. Now, \(c\), \(k\), \(c_{1}\), and \(c_{2}\) are all unknown constants so any combination of them will also be unknown constants. In particular, \(c_{1}+c_{2}k\) and \(c_{2}c\) are unknown constants so we’ll just rewrite them as follows.

\[y\left( t \right) = {c_1}{{\bf{e}}^{ - \,\frac{{b\,t}}{{2a}}}} + {c_2}\,t\,{{\bf{e}}^{ - \,\frac{{b\,t}}{{2a}}}}\]

So, if we go back to the most general form for \(v(t)\) we can take \(c=1\) and \(k=0\) and we will arrive at the same general solution.

Let’s recap. If the roots of the characteristic equation are \(r_{1} = r_{2} = r\), then the general solution is then

\[y\left( t \right) = {c_1}{{\bf{e}}^{r\,t}} + {c_2}t{{\bf{e}}^{r\,t}}\]

Now, let’s work a couple of examples.

Example 1 Solve the following IVP. \[y'' - 4y' + 4y = 0\hspace{0.25in}y\left( 0 \right) = 12\hspace{0.25in}y'\left( 0 \right) = - 3\]
Show Solution

The characteristic equation and its roots are.

\[{r^2} - 4r + 4 = {\left( {r - 2} \right)^2} = 0\hspace{0.25in}{r_{1,2}} = 2\]

The general solution and its derivative are

\[\begin{align*}y\left( t \right) & = {c_1}{{\bf{e}}^{2t}} + {c_2}t{{\bf{e}}^{2t}}\\ y'\left( t \right) & = 2{c_1}{{\bf{e}}^{2t}} + {c_2}{{\bf{e}}^{2t}} + 2{c_2}t{{\bf{e}}^{2t}}\end{align*}\]

Don’t forget to product rule the second term! Plugging in the initial conditions gives the following system.

\[\begin{align*}12 & = y\left( 0 \right) = {c_1}\\ - 3 & = y'\left( 0 \right) = 2{c_1} + {c_2}\end{align*}\]

This system is easily solve to get \(c_{1} = 12\) and \(c_{2} = -27\). The actual solution to the IVP is then.

\[y\left( t \right) = 12{{\bf{e}}^{2t}} - 27t{{\bf{e}}^{2t}}\]
Example 2 Solve the following IVP. \[16y'' - 40y' + 25y = 0\hspace{0.25in}y\left( 0 \right) = 3\hspace{0.25in}y'\left( 0 \right) = - \frac{9}{4}\]
Show Solution

The characteristic equation and its roots are.

\[16{r^2} - 40r + 25 = {\left( {4r - 5} \right)^2} = 0\hspace{0.25in}{r_{1,2}} = \frac{5}{4}\]

The general solution and its derivative are

\[\begin{align*}y\left( t \right) & = {c_1}{{\bf{e}}^{\frac{{5\,t}}{4}}} + {c_2}t{{\bf{e}}^{\frac{{5\,t}}{4}}}\\ y'\left( t \right) & = \frac{5}{4}{c_1}{{\bf{e}}^{\frac{{5\,t}}{4}}} + {c_2}{{\bf{e}}^{\frac{{5\,t}}{4}}} + \frac{5}{4}{c_2}t{{\bf{e}}^{\frac{{5\,t}}{4}}}\end{align*}\]

Don’t forget to product rule the second term! Plugging in the initial conditions gives the following system.

\[\begin{align*}3 = y\left( 0 \right) & = {c_1}\\ - \frac{9}{4} = y'\left( 0 \right) & = \frac{5}{4}{c_1} + {c_2}\end{align*}\]

This system is easily solve to get \(c_{1} = 3\) and \(c_{2} = -6\). The actual solution to the IVP is then.

\[y\left( t \right) = 3{{\bf{e}}^{\frac{{5\,t}}{4}}} - 6t{{\bf{e}}^{\frac{{5\,t}}{4}}}\]
Example 3 Solve the following IVP \[y'' + 14y' + 49y = 0\hspace{0.25in}y\left( { - 4} \right) = - 1\hspace{0.25in}y'\left( { - 4} \right) = 5\]
Show Solution

The characteristic equation and its roots are.

\[{r^2} + 14r + 49 = {\left( {r + 7} \right)^2} = 0\hspace{0.25in}{r_{1,2}} = - 7\]

The general solution and its derivative are

\[\begin{align*}y\left( t \right) & = {c_1}{{\bf{e}}^{ - 7t}} + {c_2}t{{\bf{e}}^{ - 7t}}\\ y'\left( t \right) & = - 7{c_1}{{\bf{e}}^{ - 7t}} + {c_2}{{\bf{e}}^{ - 7t}} - 7{c_2}t{{\bf{e}}^{ - 7t}}\end{align*}\]

Plugging in the initial conditions gives the following system of equations.

\[\begin{align*} - 1 & = y\left( { - 4} \right) = {c_1}{{\bf{e}}^{28}} - 4{c_2}{{\bf{e}}^{28}}\\ 5 & = y'\left( { - 4} \right) = - 7{c_1}{{\bf{e}}^{28}} + {c_2}{{\bf{e}}^{28}} + 28{c_2}{{\bf{e}}^{28}} = - 7{c_1}{{\bf{e}}^{28}} + 29{c_2}{{\bf{e}}^{28}}\end{align*}\]

Solving this system gives the following constants.

\[{c_1} = - 9{{\bf{e}}^{ - 28}}\hspace{0.25in}{c_2} = - 2{{\bf{e}}^{ - 28}}\]

The actual solution to the IVP is then.

\[\begin{align*}y\left( t \right) & = - 9{{\bf{e}}^{ - 28}}{{\bf{e}}^{ - 7t}} - 2t{{\bf{e}}^{ - 28}}{{\bf{e}}^{ - 7t}}\\ y\left( t \right) & = - 9{{\bf{e}}^{ - 7\left( {t + 4} \right)}} - 2t{{\bf{e}}^{ - 7\left( {t + 4} \right)}}\end{align*}\]
