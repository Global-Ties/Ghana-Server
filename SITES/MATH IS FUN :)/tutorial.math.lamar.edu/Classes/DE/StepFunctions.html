Section 4-4 : Step Functions

Before proceeding into solving differential equations we should take a look at one more function. Without Laplace transforms it would be much more difficult to solve differential equations that involve this function in \(g(t)\).

The function is the Heaviside function and is defined as,

\[{u_c}\left( t \right) = \left\{ {\begin{array}{*{20}{l}}0&{{\mbox{if }}t < c}\\1&{{\mbox{if }}t \ge c}\end{array}} \right.\]

Here is a graph of the Heaviside function.

Heaviside functions are often called step functions. Here is some alternate notation for Heaviside functions.

\[{u_c}\left( t \right) = u\left( {t - c} \right) = H\left( {t - c} \right)\]

We can think of the Heaviside function as a switch that is off until \(t = c\) at which point it turns on and takes a value of 1. So, what if we want a switch that will turn on and takes some other value, say 4, or -7?

Heaviside functions can only take values of 0 or 1, but we can use them to get other kinds of switches. For instance, \(4u_{c}(t)\) is a switch that is off until \(t = c\) and then turns on and takes a value of 4. Likewise, \(-7u_{c}(t)\) will be a switch that will take a value of -7 when it turns on.

Now, suppose that we want a switch that is on (with a value of 1) and then turns off at \(t = c\). We can use Heaviside functions to represent this as well. The following function will exhibit this kind of behavior.

\[1 - {u_c}\left( t \right) = \left\{ {\begin{array}{*{20}{l}}{1 - 0 = 1}&{{\mbox{if }}t < c}\\{1 - 1 = 0}&{{\mbox{if }}t \ge c}\end{array}} \right.\]

Prior to \(t = c\) the Heaviside is off and so has a value of zero. The function as whole then for \(t < c\) has a value of 1. When we hit \(t = c\) the Heaviside function will turn on and the function will now take a value of 0.

We can also modify this so that it has values other than 1 when it is on. For instance,

\[3 - 3{u_c}\left( t \right)\]

will be a switch that has a value of 3 until it turns off at \(t = c\).

We can also use Heaviside functions to represent much more complicated switches.

Example 1 Write the following function (or switch) in terms of Heaviside functions. \[f\left( t \right) = \left\{ {\begin{array}{*{20}{l}}{ - 4}&{\hspace{0.25in}{\mbox{if }}t < 6}\\{25}&{\hspace{0.25in}{\mbox{if }}6 \le t < 8}\\{16}&{\hspace{0.25in}{\mbox{if }}8 \le t < 30}\\{10}&{\hspace{0.25in}{\mbox{if }}t \ge 30}\end{array}} \right.\]
Show Solution

There are three sudden shifts in this function and so (hopefully) it’s clear that we’re going to need three Heaviside functions here, one for each shift in the function. Here’s the function in terms of Heaviside functions.

\[f\left( t \right) = - 4 + 29{u_6}\left( t \right) - 9{u_8}\left( t \right) - 6{u_{30}}\left( t \right)\]

It’s fairly easy to verify this.

In the first interval, \(t < 6\) all three Heaviside functions are off and the function has the value

\[f\left( t \right) = - 4\]

Notice that when we know that Heaviside functions are on or off we tend to not write them at all as we did in this case.

In the next interval, \(6 \le t < 8\) the first Heaviside function is now on while the remaining two are still off. So, in this case the function has the value.

\[f\left( t \right) = - 4 + 29 = 25\]

In the third interval, \(8 \le t < 30\) the first two Heaviside functions are on while the last remains off. Here the function has the value.

\[f\left( t \right) = - 4 + 29 - 9 = 16\]

In the last interval, \(t \ge 30\)all three Heaviside function are one and the function has the value.

\[f\left( t \right) = - 4 + 29 - 9 - 6 = 10\]

So, the function has the correct value in all the intervals.

All of this is fine, but if we continue the idea of using Heaviside function to represent switches, we really need to acknowledge that most switches will not turn on and take constant values. Most switches will turn on and vary continually with the value of t.

So, let’s consider the following function.

We would like a switch that is off until \(t = c\) and then turns on and takes the values above. By this we mean that when \(t = c\) we want the switch to turn on and take the value of \(f(0)\) and when t = c + 4 we want the switch to turn on and take the value of \(f(4)\), etc. In other words, we want the switch to look like the following,

Notice that in order to take the values that we want the switch to take it needs to turn on and take the values of \(f\left( {t - c} \right)\)! We can use Heaviside functions to help us represent this switch as well. Using Heaviside functions this switch can be written as

\[\begin{equation}g\left( t \right) = {u_c}\left( t \right)f\left( {t - c} \right)\label{eq:eq1}\end{equation}\]

Okay, we’ve talked a lot about Heaviside functions to this point, but we haven’t even touched on Laplace transforms yet. So, let’s start thinking about that. Let’s determine the Laplace transform of \(\eqref{eq:eq1}\). This is actually easy enough to derive so let’s do that. Plugging \(\eqref{eq:eq1}\) into the definition of the Laplace transform gives,

\[\begin{align*}\mathcal{L}\left\{ {{u_c}\left( t \right)f\left( {t - c} \right)} \right\} & = \int_{{\,0}}^{{\,\infty }}{{{{\bf{e}}^{ - s\,t}}{u_c}\left( t \right)f\left( {t - c} \right)\,dt}}\\ & = \int_{{\,c}}^{{\,\infty }}{{{{\bf{e}}^{ - s\,t}}f\left( {t - c} \right)\,dt}}\end{align*}\]

Notice that we took advantage of the fact that the Heaviside function will be zero if \(t < c\) and 1 otherwise. This means that we can drop the Heaviside function and start the integral at \(c\) instead of 0. Now use the substitution u = t – c and the integral becomes,

\[\begin{align*}\mathcal{L}\left\{ {{u_c}\left( t \right)f\left( {t - c} \right)} \right\} & = \int_{{\,0}}^{{\,\infty }}{{{{\bf{e}}^{ - s\left( {u + c} \right)}}f\left( u \right)\,du}}\\ & = \int_{{\,0}}^{{\,\infty }}{{{{\bf{e}}^{ - s\,u}}{{\bf{e}}^{ - c\,s}}f\left( u \right)\,du}}\end{align*}\]

The second exponential has no \(u\)’s in it and so it can be factored out of the integral. Note as well that in the substitution process the lower limit of integration went back to 0.

\[\mathcal{L}\left\{ {{u_c}\left( t \right)f\left( {t - c} \right)} \right\} = {{\bf{e}}^{ - c\,s}}\int_{{\,0}}^{{\,\infty }}{{{{\bf{e}}^{ - s\,u}}f\left( u \right)\,du}}\]

Now, the integral left is nothing more than the integral that we would need to compute if we were going to find the Laplace transform of f(t). Therefore, we get the following formula

\[\begin{equation}\mathcal{L}\left\{ {{u_c}\left( t \right)f\left( {t - c} \right)} \right\} = {{\bf{e}}^{ - c\,s}}F\left( s \right)\label{eq:eq2}\end{equation}\]

In order to use \(\eqref{eq:eq2}\) the function \(f(t)\) must be shifted by \(c\), the same value that is used in the Heaviside function. Also note that we only take the transform of \(f(t)\) and not f(t-c)! We can also turn this around to get a useful formula for inverse Laplace transforms.

\[\begin{equation}{\mathcal{L}^{\,\, - 1}}\left\{ {{{\bf{e}}^{ - c\,s}}F\left( s \right)} \right\} = {u_c}\left( t \right)f\left( {t - c} \right)\label{eq:eq3}\end{equation}\]

We can use \(\eqref{eq:eq2}\) to get the Laplace transform of a Heaviside function by itself. To do this we will consider the function in \(\eqref{eq:eq2}\) to be \(f(t) = 1\). Doing this gives us

\[\mathcal{L}\left\{ {{u_c}\left( t \right)} \right\} = \mathcal{L}\left\{ {{u_c}\left( t \right)\,\centerdot \,1} \right\} = {{\bf{e}}^{ - c\,s}}\mathcal{L}\left\{ 1 \right\} = \frac{1}{s}{{\bf{e}}^{ - c\,s}} = \frac{{{{\bf{e}}^{ - c\,s}}}}{s}\]

Putting all of this together leads to the following two formulas.

\[\begin{equation}\mathcal{L}\left\{ {{u_c}\left( t \right)} \right\} = \frac{{{{\bf{e}}^{ - c\,s}}}}{s}\hspace{0.25in}{\mathcal{L}^{\,\, - 1}}\left\{ {\frac{{{{\bf{e}}^{ - c\,s}}}}{s}} \right\} = {u_c}\left( t \right)\label{eq:eq4}\end{equation}\]

Let’s do some examples.

Example 2 Find the Laplace transform of each of the following.
  1. \(g\left( t \right) = 10{u_{12}}\left( t \right) + 2{\left( {t - 6} \right)^3}{u_6}\left( t \right) - \left( {7 - {{\bf{e}}^{12 - 3t}}} \right){u_4}\left( t \right)\)
  2. \(f\left( t \right) = - {t^2}{u_3}\left( t \right) + \cos \left( t \right){u_5}\left( t \right)\)
  3. \(h\left( t \right) = \left\{ {\begin{array}{*{20}{l}}{{t^4}}&{{\mbox{if }}t < 5}\\{{t^4} + 3\sin \left( {\frac{t}{{10}} - \frac{1}{2}} \right)}&{{\mbox{if }}t \ge 5}\end{array}} \right.\)
  4. \(f\left( t \right) = \left\{ {\begin{array}{*{20}{l}}t&{{\mbox{if }}t < 6}\\{ - 8 + {{\left( {t - 6} \right)}^2}}&{{\mbox{if }}t \ge 6}\end{array}} \right.\)

In all of these problems remember that the function MUST be in the form

\[{u_c}\left( t \right)f\left( {t - c} \right)\]

before we start taking transforms. If it isn’t in that form we will have to put it into that form!


a \(g\left( t \right) = 10{u_{12}}\left( t \right) + 2{\left( {t - 6} \right)^3}{u_6}\left( t \right) - \left( {7 - {{\bf{e}}^{12 - 3t}}} \right){u_4}\left( t \right)\) Show Solution

So, there are three terms in this function. The first is simply a Heaviside function and so we can use \(\eqref{eq:eq4}\) on this term. The second and third terms however have functions with them and we need to identify the functions that are shifted for each of these. In the second term it appears that we are using the following function,

\[f\left( t \right) = 2{t^3}\hspace{0.25in} \Rightarrow \hspace{0.25in}f\left( {t - 6} \right) = 2{\left( {t - 6} \right)^3}\]

and this has been shifted by the correct amount.

The third term uses the following function,

\[f\left( t \right) = 7 - {{\bf{e}}^{ - 3t}}\hspace{0.25in} \Rightarrow \hspace{0.25in}f\left( {t - 4} \right) = 7 - {{\bf{e}}^{ - 3\left( {t - 4} \right)}} = 7 - {{\bf{e}}^{12 - 3t}}\]

which has also been shifted by the correct amount.

With these functions identified we can now take the transform of the function.

\[\begin{align*}G\left( s \right) & = \frac{{10{{\bf{e}}^{ - 12s}}}}{s} + {{\bf{e}}^{ - 6s}}\frac{{2\left( {3!} \right)}}{{{s^{3 + 1}}}} - \left( {\frac{7}{s} - \frac{1}{{s + 3}}} \right){{\bf{e}}^{ - 4s}}\\ & = \frac{{10{{\bf{e}}^{ - 12s}}}}{s} + \frac{{12{{\bf{e}}^{ - 6s}}}}{{{s^{3 + 1}}}} - \left( {\frac{7}{s} - \frac{1}{{s + 3}}} \right){{\bf{e}}^{ - 4s}}\end{align*}\]

b \(f\left( t \right) = - {t^2}{u_3}\left( t \right) + \cos \left( t \right){u_5}\left( t \right)\) Show Solution

This part is going to cause some problems. There are two terms and neither has been shifted by the proper amount. The first term needs to be shifted by 3 and the second needs to be shifted by 5. So, since they haven’t been shifted, we will need to force the issue. We will need to add in the shifts, and then take them back out of course. Here they are.

\[f\left( t \right) = - {\left( {t - 3 + 3} \right)^2}{u_3}\left( t \right) + \cos \left( {t - 5 + 5} \right){u_5}\left( t \right)\]

Now we still have some potential problems here. The first function is still not really shifted correctly, so we’ll need to use

\[{\left( {a + b} \right)^2} = {a^2} + 2ab + {b^2}\]

to get this shifted correctly.

The second term can be dealt with in one of two ways. The first would be to use the formula

\[\cos \left( {a + b} \right) = \cos \left( a \right)\cos \left( b \right) - \sin \left( a \right)\sin \left( b \right)\]

to break it up into cosines and sines with arguments of \(t\)-5 which will be shifted as we expect. There is an easier way to do this one however. From our table of Laplace transforms we have #16 and using that we can see that if

\[g\left( t \right) = \cos \left( {t + 5} \right)\hspace{0.25in} \Rightarrow \hspace{0.25in}g\left( {t - 5} \right) = \cos \left( {t - 5 + 5} \right)\]

This will make our life a little easier so we’ll do it this way.

Now, breaking up the first term and leaving the second term alone gives us,

\[f\left( t \right) = - \left( {{{\left( {t - 3} \right)}^2} + 6\left( {t - 3} \right) + 9} \right){u_3}\left( t \right) + \cos \left( {t - 5 + 5} \right){u_5}\left( t \right)\]

Okay, so it looks like the two functions that have been shifted here are

\[\begin{align*}g\left( t \right) & = {t^2} + 6t + 9\\ g\left( t \right) & = \cos \left( {t + 5} \right)\end{align*}\]

Taking the transform then gives,

\[F\left( s \right) = - \left( {\frac{2}{{{s^3}}} + \frac{6}{{{s^2}}} + \frac{9}{s}} \right){{\bf{e}}^{ - 3s}} + \left( {\frac{{s\cos \left( 5 \right) - \sin \left( 5 \right)}}{{{s^2} + 1}}} \right){{\bf{e}}^{ - 5s}}\]

It’s messy, especially the second term, but there it is. Also, do not get excited about the \(\cos \left( 5 \right)\) and \(\sin \left( 5 \right)\). They are just numbers.


c \(h\left( t \right) = \left\{ {\begin{array}{*{20}{l}}{{t^4}}&{{\mbox{if }}t < 5}\\{{t^4} + 3\sin \left( {\frac{t}{{10}} - \frac{1}{2}} \right)}&{{\mbox{if }}t \ge 5}\end{array}} \right.\) Show Solution

This one isn’t as bad as it might look on the surface. The first thing that we need to do is write it in terms of Heaviside functions.

\[\begin{align*}h\left( t \right) & = {t^4} + 3{u_5}\left( t \right)\sin \left( {\frac{t}{{10}} - \frac{1}{2}} \right)\\ & = {t^4} + 3{u_5}\left( t \right)\sin \left( {\frac{1}{{10}}\left( {t - 5} \right)} \right)\end{align*}\]

Since the \(t^{4}\) is in both terms there isn’t anything to do when we add in the Heaviside function. The only thing that gets added in is the sine term. Notice as well that the sine has been shifted by the proper amount.

All we need to do now is to take the transform.

\[\begin{align*}H\left( s \right) & = \frac{{4!}}{{{s^5}}} + \frac{{3\left( {\frac{1}{{10}}} \right){{\bf{e}}^{ - 5s}}}}{{{s^2} + {{\left( {\frac{1}{{10}}} \right)}^2}}}\\ & = \frac{{24}}{{{s^5}}} + \frac{{\frac{3}{{10}}{{\bf{e}}^{ - 5s}}}}{{{s^2} + \frac{1}{{100}}}}\end{align*}\]

d \(f\left( t \right) = \left\{ {\begin{array}{*{20}{l}}t&{{\mbox{if }}t < 6}\\{ - 8 + {{\left( {t - 6} \right)}^2}}&{{\mbox{if }}t \ge 6}\end{array}} \right.\) Show Solution

Again, the first thing that we need to do is write the function in terms of Heaviside functions.

\[f\left( t \right) = t + \left( { - 8 - t + {{\left( {t - 6} \right)}^2}} \right){u_6}\left( t \right)\]

We had to add in a “-8” in the second term since that appears in the second part and we also had to subtract a \(t\) in the second term since the \(t\) in the first portion is no longer there. This subtraction of the \(t\) adds a problem because the second function is no longer correctly shifted. This is easier to fix than the previous example however.

Here is the corrected function.

\[\begin{align*}f\left( t \right) & = t + \left( { - 8 - \left( {t - 6 + 6} \right) + {{\left( {t - 6} \right)}^2}} \right){u_6}\left( t \right)\\ & = t + \left( { - 8 - \left( {t - 6} \right) - 6 + {{\left( {t - 6} \right)}^2}} \right){u_6}\left( t \right)\\ & = t + \left( { - 14 - \left( {t - 6} \right) + {{\left( {t - 6} \right)}^2}} \right){u_6}\left( t \right)\end{align*}\]

So, in the second term it looks like we are shifting

\[g\left( t \right) = {t^2} - t - 14\]

The transform is then,

\[F\left( s \right) = \frac{1}{{{s^2}}} + \left( {\frac{2}{{{s^3}}} - \frac{1}{{{s^2}}} - \frac{{14}}{s}} \right){{\bf{e}}^{ - 6s}}\]

Without the Heaviside function taking Laplace transforms is not a terribly difficult process provided we have our trusty table of transforms. However, with the advent of Heaviside functions, taking transforms can become a fairly messy process on occasion.

So, let’s do some inverse Laplace transforms to see how they are done.

Example 3 Find the inverse Laplace transform of each of the following.
  1. \(\displaystyle H\left( s \right) = \frac{{s{{\bf{e}}^{ - 4s}}}}{{\left( {3s + 2} \right)\left( {s - 2} \right)}}\)
  2. \(\displaystyle G\left( s \right) = \frac{{5{{\bf{e}}^{ - 6s}} - 3{{\bf{e}}^{ - 11s}}}}{{\left( {s + 2} \right)\left( {{s^2} + 9} \right)}}\)
  3. \(\displaystyle F\left( s \right) = \frac{{4s + {{\bf{e}}^{ - s}}}}{{\left( {s - 1} \right)\left( {s + 2} \right)}}\)
  4. \(\displaystyle G\left( s \right) = \frac{{3s + 8{{\bf{e}}^{ - 20s}} - 2s{{\bf{e}}^{ - 3s}} + 6{{\bf{e}}^{ - 7s}}}}{{{s^2}\left( {s + 3} \right)}}\)

All of these will use \(\eqref{eq:eq3}\) somewhere in the process. Notice that in order to use this formula the exponential doesn’t really enter into the mix until the very end. The vast majority of the process is finding the inverse transform of the stuff without the exponential.

In these problems we are not going to go into detail on many of the inverse transforms. If you need a refresher on some of the basics of inverse transforms go back and take a look at the previous section.


a \(\displaystyle H\left( s \right) = \frac{{s{{\bf{e}}^{ - 4s}}}}{{\left( {3s + 2} \right)\left( {s - 2} \right)}}\) Show Solution

In light of the comments above let’s first rewrite the transform in the following way.

\[H\left( s \right) = {{\bf{e}}^{ - 4s}}\frac{s}{{\left( {3s + 2} \right)\left( {s - 2} \right)}} = {{\bf{e}}^{ - 4s}}F\left( s \right)\]

Now, this problem really comes down to needing \(f(t)\). So, let’s do that. We’ll need to partial fraction \(F(s)\) up. Here’s the partial fraction decomposition.

\[F\left( s \right) = \frac{A}{{3s + 2}} + \frac{B}{{s - 2}}\]

Setting numerators equal gives,

\[s = A\left( {s - 2} \right) + B\left( {3s + 2} \right)\]

We’ll find the constants here by selecting values of \(s\). Doing this gives,

\[\begin{align*} & s = 2 & 2 & = 8B & \Rightarrow \hspace{0.25in}B & = \frac{1}{4}\\ & s = - \frac{2}{3} & - \frac{2}{3} & = - \frac{8}{3}A & \Rightarrow \hspace{0.25in}A & = \frac{1}{4}\end{align*}\]

So, the partial fraction decomposition becomes,

\[F\left( s \right) = \frac{{\frac{1}{4}}}{{3\left( {s + \frac{2}{3}} \right)}} + \frac{{\frac{1}{4}}}{{s - 2}}\]

Notice that we factored a 3 out of the denominator in order to actually do the inverse transform. The inverse transform of this is then,

\[f\left( t \right) = \frac{1}{{12}}{{\bf{e}}^{ - \frac{{2\,t}}{3}}} + \frac{1}{4}{{\bf{e}}^{2t}}\]

Now, let’s go back and do the actual problem. The original transform was,

\[H\left( s \right) = {{\bf{e}}^{ - 4s}}F\left( s \right)\]

Note that we didn’t bother to plug in \(F(s)\). There really isn’t a reason to plug it back in. Let’s just use \(\eqref{eq:eq3}\) to write down the inverse transform in terms of symbols. The inverse transform is,

\[h\left( t \right) = {u_4}\left( t \right)f\left( {t - 4} \right)\]

where, \(f(t)\) is,

\[f\left( t \right) = \frac{1}{{12}}{{\bf{e}}^{ - \frac{{2\,t}}{3}}} + \frac{1}{4}{{\bf{e}}^{2t}}\]

This is all the farther that we’ll go with the answer. There really isn’t any reason to plug in \(f(t)\) at this point. It would make the function longer and definitely messier. We will give almost all of our answers to these types of inverse transforms in this form.


b \(\displaystyle G\left( s \right) = \frac{{5{{\bf{e}}^{ - 6s}} - 3{{\bf{e}}^{ - 11s}}}}{{\left( {s + 2} \right)\left( {{s^2} + 9} \right)}}\) Show Solution

This problem is not as difficult as it might at first appear to be. Because there are two exponentials we will need to deal with them separately eventually. Now, this might lead us to conclude that the best way to deal with this function is to split it up as follows,

\[G\left( s \right) = {{\bf{e}}^{ - 6s}}\frac{5}{{\left( {s + 2} \right)\left( {{s^2} + 9} \right)}} - {{\bf{e}}^{ - 11s}}\frac{3}{{\left( {s + 2} \right)\left( {{s^2} + 9} \right)}}\]

Notice that we factored out the exponential, as we did in the last example, since we would need to do that eventually anyway. This is where a fairly common complication arises. Many people will call the first function \(F(s)\) and the second function \(H(s)\) and then partial fraction both of them.

However, if instead of just factoring out the exponential we would also factor out the coefficient we would get,

\[G\left( s \right) = 5{{\bf{e}}^{ - 6s}}\frac{1}{{\left( {s + 2} \right)\left( {{s^2} + 9} \right)}} - 3{{\bf{e}}^{ - 11s}}\frac{1}{{\left( {s + 2} \right)\left( {{s^2} + 9} \right)}}\]

Upon doing this we can see that the two functions are in fact the same function. The only difference is the constant that was in the numerator. So, the way that we’ll do these problems is to first notice that both of the exponentials have only constants as coefficients. Instead of breaking things up then, we will simply factor out the whole numerator and get,

\[G\left( s \right) = \left( {5{{\bf{e}}^{ - 6s}} - 3{{\bf{e}}^{ - 11s}}} \right)\frac{1}{{\left( {s + 2} \right)\left( {{s^2} + 9} \right)}} = \left( {5{{\bf{e}}^{ - 6s}} - 3{{\bf{e}}^{ - 11s}}} \right)F\left( s \right)\]

and now we will just partial fraction \(F(s)\).

Here is the partial fraction decomposition.

\[F\left( s \right) = \frac{A}{{s + 2}} + \frac{{Bs + C}}{{{s^2} + 9}}\]

Setting numerators equal and combining gives us,

\[\begin{align*}1 & = A\left( {{s^2} + 9} \right) + \left( {s + 2} \right)\left( {Bs + C} \right)\\ & = \left( {A + B} \right){s^2} + \left( {2B + C} \right)s + 9A + 2C\end{align*}\]

Setting coefficient equal and solving gives,

\[\left. {\begin{align*} & {s^2}\,: & A + B & = 0\\& {s^1}\,: & 2B + C & = 0\\& {s^0}\,: & 9A + 2C & = 1\end{align*}} \right\}\hspace{0.25in} \Rightarrow \hspace{0.25in}A = \frac{1}{{13}},\,\,\,\,B = - \frac{1}{{13}},\,\,\,\,\,C = \frac{2}{{13}}\]

Substituting back into the transform gives and fixing up the numerators as needed gives,

\[\begin{align*}F\left( s \right) &= \frac{1}{{13}}\left( {\frac{1}{{s + 2}} + \frac{{ - s + 2}}{{{s^2} + 9}}} \right)\\ & = \frac{1}{{13}}\left( {\frac{1}{{s + 2}} - \frac{s}{{{s^2} + 9}} + \frac{{2\frac{3}{3}}}{{{s^2} + 9}}} \right)\end{align*}\]

As we did in the previous section we factored out the common denominator to make our work a little simpler. Taking the inverse transform then gives,

\[f\left( t \right) = \frac{1}{{13}}\left( {{{\bf{e}}^{ - 2t}} - \cos \left( {3t} \right) + \frac{2}{3}\sin \left( {3t} \right)} \right)\]

At this point we can go back and start thinking about the original problem.

\[\begin{align*}G\left( s \right) & = \left( {5{{\bf{e}}^{ - 6s}} - 3{{\bf{e}}^{ - 11s}}} \right)F\left( s \right)\\ & = 5{{\bf{e}}^{ - 6s}}F\left( s \right) - 3{{\bf{e}}^{ - 11s}}F\left( s \right)\end{align*}\]

We’ll also need to distribute the \(F(s)\) through as well in order to get the correct inverse transform. Recall that in order to use \(\eqref{eq:eq3}\) to take the inverse transform you must have a single exponential times a single transform. This means that we must multiply the \(F(s)\) through the parenthesis. We can now take the inverse transform,

\[g\left( t \right) = 5{u_6}\left( t \right)f\left( {t - 6} \right) - 3{u_{11}}\left( t \right)f\left( {t - 11} \right)\]

where,

\[f\left( t \right) = \frac{1}{{13}}\left( {{{\bf{e}}^{ - 2t}} - \cos \left( {3t} \right) + \frac{2}{3}\sin \left( {3t} \right)} \right)\]

c \(\displaystyle F\left( s \right) = \frac{{4s + {{\bf{e}}^{ - s}}}}{{\left( {s - 1} \right)\left( {s + 2} \right)}}\) Show Solution

In this case, unlike the previous part, we will need to break up the transform since one term has a constant in it and the other has an \(s\). Note as well that we don’t consider the exponential in this, only its coefficient. Breaking up the transform gives,

\[F\left( s \right) = \frac{{4s}}{{\left( {s - 1} \right)\left( {s + 2} \right)}} + {{\bf{e}}^{ - s}}\frac{1}{{\left( {s - 1} \right)\left( {s + 2} \right)}} = G\left( s \right) + {{\bf{e}}^{ - s}}H\left( s \right)\]

We will need to partial fraction both of these terms up. We’ll start with \(G(s)\).

\[G\left( s \right) = \frac{A}{{s - 1}} + \frac{B}{{s + 2}}\]

Setting numerators equal gives,

\[4s = A\left( {s + 2} \right) + B\left( {s - 1} \right)\]

Now, pick values of \(s\) to find the constants.

\[\begin{align*} & s = - 2 & - 8 & = - 3B & \Rightarrow \hspace{0.25in}B & = \frac{8}{3}\\ & s = 1 & 4 & = 3A & \Rightarrow \hspace{0.25in}A &= \frac{4}{3}\end{align*}\]

So \(G(s)\) and its inverse transform is,

\[\begin{align*}G\left( s \right) & = \frac{{\frac{4}{3}}}{{s - 1}} + \frac{{\frac{8}{3}}}{{s + 2}}\\ g\left( t \right) & = \frac{4}{3}{{\bf{e}}^t} + \frac{8}{3}{{\bf{e}}^{ - 2t}}\end{align*}\]

Now, repeat the process for \(H(s)\).

\[H\left( s \right) = \frac{A}{{s - 1}} + \frac{B}{{s + 2}}\]

Setting numerators equal gives,

\[1 = A\left( {s + 2} \right) + B\left( {s - 1} \right)\]

Now, pick values of \(s\) to find the constants.

\[\begin{align*}& s = - 2 & 1 & = - 3B & \Rightarrow \hspace{0.25in}B & = - \frac{1}{3}\\ & s = 1 & 1 & = 3A & \Rightarrow \hspace{0.25in}A & = \frac{1}{3}\end{align*}\]

So \(H(s)\) and its inverse transform is,

\[\begin{align*}H\left( s \right) & = \frac{{\frac{1}{3}}}{{s - 1}} - \frac{{\frac{1}{3}}}{{s + 2}}\\ h\left( t \right) & = \frac{1}{3}{{\bf{e}}^t} - \frac{1}{3}{{\bf{e}}^{ - 2t}}\end{align*}\]

Putting all of this together gives the following,

\[\begin{align*}F\left( s \right) & = G\left( s \right) + {{\bf{e}}^{ - s}}H\left( s \right)\\ f\left( t \right) & = g\left( t \right) + {u_1}\left( t \right)h\left( {t - 1} \right)\end{align*}\]

where,

\[g\left( t \right) = \frac{4}{3}{{\bf{e}}^t} + \frac{8}{3}{{\bf{e}}^{ - 2t}}\hspace{0.25in}{\mbox{and}}\hspace{0.25in}h\left( t \right) = \frac{1}{3}{{\bf{e}}^t} - \frac{1}{3}{{\bf{e}}^{ - 2t}}\]

d \(\displaystyle G\left( s \right) = \frac{{3s + 8{{\bf{e}}^{ - 20s}} - 2s{{\bf{e}}^{ - 3s}} + 6{{\bf{e}}^{ - 7s}}}}{{{s^2}\left( {s + 3} \right)}}\) Show Solution

This one looks messier than it actually is. Let’s first rearrange the numerator a little.

\[G\left( s \right) = \frac{{s\left( {3 - 2{{\bf{e}}^{ - 3s}}} \right) + \left( {8{{\bf{e}}^{ - 20s}} + 6{{\bf{e}}^{ - 7s}}} \right)}}{{{s^2}\left( {s + 3} \right)}}\]

In this form it looks like we can break this up into two pieces that will require partial fractions. When we break these up we should always try and break things up into as few pieces as possible for the partial fractioning. Doing this can save you a great deal of unnecessary work. Breaking up the transform as suggested above gives,

\[\begin{align*}G\left( s \right) & = \left( {3 - 2{{\bf{e}}^{ - 3s}}} \right)\frac{1}{{s\left( {s + 3} \right)}} + \left( {8{{\bf{e}}^{ - 20s}} + 6{{\bf{e}}^{ - 7s}}} \right)\frac{1}{{{s^2}\left( {s + 3} \right)}}\\ & = \left( {3 - 2{{\bf{e}}^{ - 3s}}} \right)F\left( s \right) + \left( {8{{\bf{e}}^{ - 20s}} + 6{{\bf{e}}^{ - 7s}}} \right)H\left( s \right)\end{align*}\]

Note that we canceled an \(s\) in \(F(s)\). You should always simplify as much a possible before doing the partial fractions.

Let’s partial fraction up \(F(s)\) first.

\[F\left( s \right) = \frac{A}{s} + \frac{B}{{s + 3}}\]

Setting numerators equal gives,

\[1 = A\left( {s + 3} \right) + Bs\]

Now, pick values of \(s\) to find the constants.

\[\begin{align*} & s = - 3 & 1 & = - 3B & \Rightarrow \hspace{0.25in}B & = - \frac{1}{3}\\ & s = 0 & 1 & = 3A & \Rightarrow \hspace{0.25in}A &= \frac{1}{3}\end{align*}\]

So \(F(s)\) and its inverse transform is,

\[\begin{align*}F\left( s \right) & = \frac{{\frac{1}{3}}}{s} - \frac{{\frac{1}{3}}}{{s + 3}}\\ f\left( t \right) & = \frac{1}{3} - \frac{1}{3}{{\bf{e}}^{ - 3t}}\end{align*}\]

Now partial fraction \(H(s)\).

\[H\left( s \right) = \frac{A}{s} + \frac{B}{{{s^2}}} + \frac{C}{{s + 3}}\]

Setting numerators equal gives,

\[1 = As\left( {s + 3} \right) + B\left( {s + 3} \right) + C{s^2}\]

Pick values of \(s\) to find the constants.

\[\begin{align*}& s = - 3 & 1 & = 9C & \Rightarrow \hspace{0.25in}C & = \frac{1}{9}\\ & s = 0 & 1 & = 3B & \Rightarrow \hspace{0.25in}B & = \frac{1}{3}\\ & s = 1 & 1 & = 4A + 4B + C = 4A + \frac{{13}}{9} & \Rightarrow \hspace{0.25in}A &= - \frac{1}{9}\end{align*}\]

So, \(H(s)\) and its inverse transform is,

\[\begin{align*}H\left( s \right) & = - \frac{{\frac{1}{9}}}{s} + \frac{{\frac{1}{3}}}{{{s^2}}} + \frac{{\frac{1}{9}}}{{s + 3}}\\ h\left( t \right) & = - \frac{1}{9} + \frac{1}{3}t + \frac{1}{9}{{\bf{e}}^{ - 3t}}\end{align*}\]

Now, let’s go back to the original problem, remembering to multiply the transform through the parenthesis.

\[G\left( s \right) = 3F\left( s \right) - 2{{\bf{e}}^{ - 3s}}F\left( s \right) + 8{{\bf{e}}^{ - 20s}}H\left( s \right) + 6{{\bf{e}}^{ - 7s}}H\left( s \right)\]

Taking the inverse transform gives,

\[g\left( t \right) = 3f\left( t \right) - 2{u_3}\left( t \right)f\left( {t - 3} \right) + 8{u_{20}}\left( t \right)h\left( {t - 20} \right) + 6{u_7}\left( t \right)h\left( {t - 7} \right)\]

So, as this example has shown, these can be a somewhat messy. However, the mess is really only that of notation and amount of work. The actual partial fraction work was identical to the previous sections work. The main difference in this section is we had to do more of it. As far as the inverse transform process goes. Again, the vast majority of that was identical to the previous section as well.

So, don’t let the apparent messiness of these problems get you to decide that you can’t do them. Generally, they aren’t as bad as they seem initially.
